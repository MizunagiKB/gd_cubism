About GDCubism

このソフトウェア（以下GDCubism）は Live2D Cubism SDKを GDScript や C# から利用可能にするための非公式な GDExtension です。

この GDExtension は Godot Engine 4.1.1 以降で使用可能です。

GDCubism 内部ではLive2D社が公開している Cubism Native Framework を併用しています。

GDCubism については開発元のLive2D社に問い合わせない様お願い致します。
