デバッグ方法

GDCubism のデバッグを行うには VisualStudio Code を使用して以下の方法で行えます。

  1. SCons のビルド時に debug_symbols=true optimize=none を追加します。

  2. launch.json に以下を追加します。

  3. Godot で demo プロジェクトを開き Export を行います。

    1. 作業がしやすい様に、 demo.export というフォルダが事前に用意されています。

  4. デバッグ実行を行います。

Windows

{
    "name": "Debug (Windows)",
    "type": "cppvsdbg",
    "request": "launch",
    "program": "${workspaceFolder}\\demo.export\\demo.exe",
    "externalConsole": true,
    "logging": {
        "moduleLoad": false,
        "trace": true
    },
    "cwd": "${workspaceFolder}\\demo.export",
}

macOS

{
    "name": "Debug (macOS)",
    "type":"cppdbg",
    "request": "launch",
    "program": "${workspaceRoot}/demo.export/demo.app/Contents/MacOS/demo",
    "stopAtEntry": false,
    "launchCompleteCommand": "exec-run",
    "osx": {
        "MIMode": "lldb"
    },
    "cwd": "${workspaceRoot}/demo.export",
}
